서버에서 데이터를 공유, 저장하는 객체

  • HttpServletRequest : 1회용 데이터를 저장하는 저장소 요청하고 응답이 완료될때까지만 사용 * 일반적으로 가장 많이 사용하는 저장소
  • HttpSession : 원하는 시점까지 데이터를 유지하는 저장소 개발자가 삭제하거나 설정값에 의해 자동으로 삭제될때까지 이용
  • ServletContext : 서버의 context영역이 생성되면 유지 서버시작되면 유지, 종료시 삭제;
